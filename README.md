# 🎹 Proiect Design Orgă Electronică (Digital Keyboard) - AutoCAD

## 📝 Descriere Proiect
Acest repository conține proiectul detaliat al unei **orgi electronice** realizat integral în **AutoCAD**. Proiectul a fost conceput pentru a demonstra precizia în modelare, atenția la detalii ergonomice și organizarea riguroasă a layerelor într-un context de design industrial/muzical.

Modelul cuprinde elemente esențiale precum:
* **Claviatură completă** (clape albe și negre cu profil realist).
* **Panou de control** (afișaj LCD, butoane rotative, slidere de volum).
* **Sistem audio** (grilaje difuzoare modelate prin pattern-uri repetitive).
* **Conectică spate** (porturi MIDI, USB, Jack, Power).

---

## 🛠 Detalii Tehnice
* **Software utilizat:** AutoCAD (Versiunea 2024)
* **Unități de măsură:** Milimetri (mm)
* **Format fișier:** `.dwg` (Compatibil cu AutoCAD 2018+)
* **Organizare:** Layere separate pentru carcasă, clape, componente electronice și texte.

---

## 📂 Structura Repository-ului
* `/dwg` - Fișierele sursă AutoCAD.
* `/pdf` - Planșe tehnice exportate pentru vizualizare rapidă.
* `/img` - Screenshot-uri și randări ale modelului.

---

## 🚀 Vizualizare și Utilizare
1. **Clonează repository-ul:** `git clone https://github.com/utilizator/nume-proiect.git`
2. **Deschide fișierul:** Deschide `orga_electronica.dwg` folosind AutoCAD sau un viewer compatibil (DWG TrueView).
3. **Setări Recomandate:** Pentru o experiență optimă, folosește stilul vizual **"Realistic"** sau **"Shaded with Edges"**.
